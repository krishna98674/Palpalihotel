# 🍽 Palpali Hotel Management System

## Project Description

Palpali Hotel Management System is a web-based application developed using Node.js, Express.js, MySQL, HTML, CSS, and JavaScript. It allows customers to browse food items, reserve tables, and place food orders. Administrators can manage foods, reservations, and customer orders through an admin dashboard.

---

## Features

- Home Page
- Dynamic Food Menu
- Shopping Cart
- Checkout System
- Reservation System
- Admin Dashboard
- Food Management
- Order Management
- Reservation Management
- MySQL Database Integration

## Installation

```bash
git clone <repository-url>
```

```bash
cd Palpalihotel
```

```bash
npm install
```

Create `.env`

```env
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=your_password
DB_NAME=palpalihotel
PORT=5000
```

Run

```bash
npm start
```

Open

```
http://localhost:5000
```
